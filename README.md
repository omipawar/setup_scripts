# setup_scripts

## Terraform Installation script
```bash
curl -O https://raw.githubusercontent.com/omipawar/setup_scripts/main/install-terraform.sh
chmod +x install-terraform.sh
./install-terraform.sh
```
### OR
```bash
### curl -s https://raw.githubusercontent.com/omipawar/setup_scripts/main/install-terraform.sh | bash
```

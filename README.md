# setup_scripts

## Terraform Installation script
```bash
curl -O https://raw.githubusercontent.com/omipawar/setup_scripts/refs/heads/main/install_terraform.sh
chmod +x install_terraform.sh
./install_terraform.sh
```
### OR
```bash
### curl -s https://raw.githubusercontent.com/omipawar/setup_scripts/refs/heads/main/install_terraform.sh | bash
```
